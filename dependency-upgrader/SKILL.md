## What I do
- Scan dependencies for outdated, vulnerable, or deprecated packages
- Create safe upgrade plans with change impact analysis
- Handle breaking changes through codemods and compatibility layers
- Update lockfiles while preserving reproducible builds
- Test upgrades against existing test suites
- Generate changelogs for dependency updates
- Support npm, yarn, pnpm, pip, go mod, cargo

## When to use me
Use this skill when:
- Security advisories affect your dependencies
- Needing to upgrade to a new major version
- Routine dependency maintenance windows
- Preparing releases and need updated dependencies
- Resolving version conflicts between packages
- Auditing dependencies for license compliance
