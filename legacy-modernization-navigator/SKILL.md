## What I do

- Analyze current technology stack and dependencies
- Identify outdated libraries, deprecated APIs, and security vulnerabilities
- Create phased migration plans with minimal risk
- Recommend compatible modern alternatives
- Generate migration checkpoints and rollback strategies
- Assess migration complexity and estimated effort

## When to use me

Use this skill when:
- Inheriting a legacy codebase with unknown dependencies
- Planning a technology upgrade (frameworks, languages, platforms)
- Evaluating modernization options and trade-offs
- Creating a roadmap for incremental migration
- Need to convince stakeholders of migration priorities
- Want to avoid big-bang rewrites in favor of incremental changes
